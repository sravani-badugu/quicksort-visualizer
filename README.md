# QuickSort Algorithm Visualizer

A premium, interactive QuickSort algorithm visualizer featuring a modern dark-themed glassmorphism UI, responsive design, and smooth animations.

## Features

- **Interactive Visualization**: Step-by-step sorting logic with real-time tracking of comparisons and swaps.
- **Custom Input**: Generate random arrays or input custom comma-separated values.
- **Playback Controls**: Start, Pause, Resume, and Reset the visualization at any time.
- **Speed Control**: Dynamically adjust the sorting speed during execution.
- **Detailed Statistics**: Track comparisons, swaps, execution time, and algorithmic complexity (Time: O(N log N), Space: O(log N)).
- **Action Log**: Real-time "dry run" action log explaining the algorithm's current state.
- **Modern UI/UX**: Premium aesthetic with dark-theme, glassmorphic panels, glowing elements, and responsive layout.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox/Grid layouts, Glassmorphism effects, responsive design.
- **Vanilla JavaScript**: DOM manipulation, asynchronous sorting logic (Promises & async/await), state management.

## How to Run

1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. No build tools or dependencies are required.

## Usage

- Click **Generate Random (10)** to create a new array.
- Enter custom numbers in the input field and click **Load Custom**.
- Click **Start Sorting** to begin the visualization.
- Use the **Sorting Speed** slider to control animation speed.
- Observe the algorithm's progress, statistics, and action log.
